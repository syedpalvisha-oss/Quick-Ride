# Quick-Ride 🚗

## Problem Statement
Urban ride coordination is inefficient when managing multiple riders, drivers, and scheduling constraints manually. There is a need for a structured system to streamline ride allocation and tracking.

## Solution
Quick-Ride is a ride scheduling and logistics platform that enables efficient coordination between riders and drivers through a structured backend system.

## Features
- Ride booking and scheduling
- Driver assignment system
- Ride history tracking
- Role-based access control (Admin, Driver, User)
- Modular Laravel architecture

## Unique Aspects
- Structured ride flow (request → assignment → completion)
- Scalable backend design for logistics use cases
- Designed for future real-time tracking integration

## Tech Stack
- Backend: Laravel (PHP)
- Database: MySQL
- Frontend: Blade Templates
- Architecture: MVC

## How to Run
1. Clone repo
2. Run `composer install`
3. Configure `.env`
4. Run `php artisan migrate`
5. Start server: `php artisan serve`
