# Quick-Ride

## Overview
AI-powered ride scheduling and logistics platform designed to streamline ride management with driver and rider modules.

## Features
- Ride booking and scheduling
- Driver assignment system
- Ride history tracking
- Role-based access (Admin, Driver, User)
- Future scope: real-time tracking & map integration

## Tech Stack
- Backend: Laravel (PHP)
- Database: MySQL
- Frontend: Blade Templates

## Status
In development – core features implemented
